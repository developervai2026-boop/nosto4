# 🤖 নস্টো বাবা - Firebase AI Chat & Image Generator

A Bengali/English AI chatbot with image generation powered by Firebase Vertex AI (Gemini + Imagen).

## ✨ Features
- 💬 AI Chat with streaming responses (typing effect)
- 🎨 Image generation with Imagen 3.0
- 🌐 বাংলা & English language support
- 💾 Chat history saved to Firestore
- 🔐 App Check with reCAPTCHA v3 for security
- 📱 Responsive mobile-friendly UI
